# 🧮 Scientific Calculator

A fully functional **Scientific Calculator** built using **HTML, CSS (Bootstrap), and JavaScript**.

---

## 🚀 Features

### ✅ Basic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Percentage (%)

### 🔬 Scientific Functions
- sin, cos, tan (Degree Mode)
- log (base 10)
- ln (natural log)
- √ (square root)
- π and e constants

### ⚡ Advanced Features
- Power functions (x², xʸ)
- Keyboard support (type using keyboard)
- Backspace (⌫)
- Clear (AC / C)
- Error handling

---

👉 **Degree Mode Enabled**
